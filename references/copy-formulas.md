# 카피 공식 라이브러리 (재사용) — R1

> 카드뉴스 자동생성 시 hook/body/CTA 슬롯에 주입할 검증된 템플릿.
> `{}` = 변수 슬롯. 글자수 가드(라이브웹∩codex): 표지 메인훅 ≤24자, 서브헤드 ≤34자, 본문 카드 ≤90자, CTA메인 ≤14자.

## A. 표지 훅 템플릿 (10 유형)

| # | 유형 | 템플릿 | 예시 |
|---|---|---|---|
| H1 | 숫자형 | `{주제} {N}가지 {이득}` | "협상 잘하는 7가지 말버릇" |
| H2 | 손실회피 | `{대상}이면 이거 모르면 손해` | "직장인이면 이 연말정산 모르면 손해" |
| H3 | 질문형 | `왜 {의외의 사실}일까?` / `{A} vs {B}, 뭐가 맞을까?` | "왜 부자는 대출을 더 많이 쓸까?" |
| H4 | 방법형 | `{제약} 없이 {목표}하는 법` | "코딩 없이 카드뉴스 만드는 법" |
| H5 | 반전/미신 | `{통념}? 사실은 {반전}` | "물 8잔? 사실은 근거 없습니다" |
| H6 | 금지형 | `{흔한 실수} 하지 마세요` | "이력서에 이 단어 쓰지 마세요" |
| H7 | 징후/체크리스트 | `{상태}의 {N}가지 신호` | "번아웃의 5가지 신호" |
| H8 | 전후비교 | `{Before} → {After}, 차이는 {키}` | "월급 그대로인데 통장 잔고가 바뀐 이유" |
| H9 | 타겟지정 | `{N년차/특정독자}라면 꼭` | "3년차 마케터라면 꼭 아는 지표" |
| H10 | 호기심갭 | `{결과}, 그 이유가 의외였다` | "매출 2배 오른 가게, 바꾼 건 딱 하나" |

훅 원칙: ①구체 숫자·고유명사 넣기 ②표지에 답 다 쓰지 않기(스와이프 유인) ③1줄 25자 이내 ④"관련성+가치"를 즉시 신호.

## B. 2번 슬라이드(신뢰/예고) 템플릿
- `{권위/데이터}에 따르면...` (신뢰)
- `이 글 다 보면 {얻는 것} 알게 됩니다` (예고)
- `순서대로 {N}개만 보세요` (구조 예고)

## C. 본문 카드 템플릿 (1카드=1메시지, ≤80자)
- 주장 → 근거 1개: `{한 문장 주장}. 왜냐면 {근거}.`
- 숫자 강조: `{수치}. {이 숫자의 의미 한 줄}.`
- 대조: `남들은 {A}. 하지만 {B}.`
- 단계: `STEP {n}. {행동 한 줄}.`

## D. CTA 템플릿
| 목적 | 템플릿 |
|---|---|
| 저장(Save) | "다음에 {상황}일 때 위해 저장 ↓" |
| 공유(Send) | "{대상}에게 이거 보내주세요" |
| DM | "'{키워드}' DM 주시면 {보상} 드려요" |
| 팔로우 | (캡션 끝줄) "더 알고싶다면 팔로우 @{핸들}" |
| 요약형 마지막장 | "오늘의 핵심 {N}줄 요약 + @{핸들}·로고" ← 한국형 최적 |

배치: 중간(5~6번) 소프트 CTA 1회 + 마지막장 명확 CTA 1회.

## E. 뉴스 → 카드 변환 규칙 (파이프라인 룰)
1. 뉴스에서 **팩트 리스트 추출**(수치·고유명사·인과·시점) → 절대 누락/왜곡 금지.
2. **표지** = 팩트 중 가장 의외/충격적인 1개를 호기심갭 훅(H1/H2/H5/H10)으로. 답은 감춘다.
3. **본문 배분** = 팩트 1개 = 카드 1장. 6~8장 목표.
4. **배경 컨텍스트**(뉴닉식)를 2번 카드에 먼저, 전문용어는 일상어로 치환.
5. **요약 카드**(끝-1) = 핵심 팩트 N줄 재나열(저장 가치 극대화).
6. **CTA 카드**(끝) = 요약+저장유도. 팔로우는 캡션.
7. **사실/해석/전망/미확인 라벨 분리**(codex§6.3): 사실=공식발표·통계, 해석=전문가의견, 전망=향후일정, 미확인=출처충돌. 본문 문장에 톤으로 구분.
8. **숫자엔 단위+기준일**: "3.5%" 금지 → "작년 2.1%→올해 3.5%(2026.6 기준)".
9. 검증: 표지 메인훅 ≤24자, 본문 ≤90자/카드, 팩트 보존 100%, 사실≠해석 미혼동.

## F. 피해야 할 훅(금지 목록, codex§5.5)
- "충격/대박/무조건/절대" 남발 · 출처 없이 공포 조장 · 기사 제목 그대로 베끼기 · 본문이 못 지키는 과장 약속 · 모든 대상에게 말하는 범용 제목("요즘 꼭 알아야 할" → "전세 사는 사람이 이번 달 확인할").

## G. 카드→쇼츠 전환 유지값 (R2/S7 핸드오프, codex§10)
- 표지 훅 = 영상 0~2초 첫 자막/내레이션.
- **카드 1장 = 쇼츠 1비트**. 7장 카드뉴스 ≈ 20~35초 쇼츠.
- 슬라이드 제목 = 쇼츠 자막 첫 줄. 본문은 내레이션, 화면엔 키워드만.
- 저장형은 쇼츠도 끝에 "저장" CTA, 공유형은 "이 사람에게 보내기"로 수신자 특정.
